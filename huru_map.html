HURUmap - Kenya
Provides easy access to health, census and demographics data.
Introduction

HURUmap helps journalists add context and depth to data stories by giving them easy and quick access to reliable and factual data. HURUmap takes data from the country's census databases and turns it into a mobile-friendly website with simple data visualizations of county, city and district level data on key demographic issues. These data visualizations are embeddable so that journalists can use them in their stories. HURUmap also allows for the comparison of any two areas side-by-side to see what the data reveals about those regions. HURUmap is based on WAZImap, which Media Monitoring Africa and Code for South Africa developed as a tool to provide easy access to South African census and elections data. Code for Africa is expanding HURUmap to Tanzania and Zambia. WAZImap is built on the U.S.-based Census Reporter.
